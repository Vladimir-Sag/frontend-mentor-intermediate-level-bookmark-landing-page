## 📝 Project Overview

A responsive landing page for a bookmark management application. Users can learn about features, browser extensions, and sign up for newsletter updates. Built with semantic HTML5, CSS3, and vanilla JavaScript.

**This project was completed as part of the [Frontend Mentor](https://www.frontendmentor.io/challenges/bookmark-landing-page-5d0b588a9edda32581d29158) challenge.**

---

## ✨ Features

- ✅ Fully responsive layout (mobile → tablet → desktop)
- ✅ Mobile navigation menu with smooth overlay
- ✅ Interactive tabs for feature sections
- ✅ FAQ accordion with expand/collapse functionality
- ✅ Newsletter form with email validation
- ✅ Error messages for empty/invalid email input
- ✅ Hover states for all interactive elements
- ✅ Rounded blue background shapes created with pure CSS
- ✅ SVG icon color changes on hover
- ✅ Accessible with semantic HTML and ARIA labels

## 🧩 Key Features Implementation

### Mobile Menu
- Hamburger button opens full-screen overlay navigation
- Smooth transition animations
- Close button and link clicks close the menu

### Interactive Tabs
- Three feature tabs (Simple Bookmarking, Speedy Searching, Easy Sharing)
- Active tab highlighted with red bottom border
- Content changes dynamically without page reload

### FAQ Accordion
- Click on questions to expand/collapse answers
- Chevron icon rotates on open/close
- Smooth height transitions

### Newsletter Form Validation
- Validates email format using regex
- Shows custom error message for:
  - Empty input field
  - Invalid email format
- Prevents form submission on validation errors
